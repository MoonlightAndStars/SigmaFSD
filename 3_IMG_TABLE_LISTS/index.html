HTML Images

To include images on any webpage we must use the img tag with attribute src and alt. src attribute is mandatory and alt attribute is better for SEO practices.

We can set either height or width in img attribute for SEO purposes. It is by default in pixel and we do not need to write px or pixel after giving value.

They are very important in HTML. There are three types of lists in HTML.

  1. Ordered List
  2. Unordered List
  3. Definition List - less likely to be used but important to know.

The Ordered List and Unordered List are mostly used and are very important.

Ordered List

An ordered list is created by ol tag and has different list styles. The list items are created by using the li tag. There are different ways to arrange an ordered list. By default they are numbered but they can be converted to roman numbers, small-cap roman, small cap alphabets.

  1. By default the list type is numbered.
  2. if we use type="i" attribute then the list will be numbered in small roman numbers
  3. if we use type="I" attribute then the list will be numbered in caps roman numbers
  4. if we use type="a" attribute then the list will be numbered in small cap aplhabets starting from a
  5. if we use type="A" attribute then the list will be numbered in large cap alphabets.
  6. we can also set a start position of our list in order to make our list start from a particular number or letter instead of default starting points. in the main ol tag we need to give start attribute with the start location

Unordered List

An unordered list is created by using ul tag and has list styles are circle, square, none, etc.The list items are created by using the li tag.
