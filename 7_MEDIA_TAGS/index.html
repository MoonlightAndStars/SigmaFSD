MEDIA- AUDIO and VIDEO

Video Tag

We use %lt;video> tag to add the video in our website.

Attributes of a video tag.

  1. src = src attribute of the HTML video tag is primarily designed to point to video files that are:
    • hosted on the same server as our website.
    • accessible via a direct URL to a video file (e.g., a .mp4, .webm, or .ogg file).
    • therefore, we generally use the src attribute to link to video files that we have uploaded and stored yourself.
  2. controls = to add controls on the video. if controls not added, the video will not be playable. after adding controls, we can make play the video, pause the video, mute the video or even make video full-screen, download it, mute it, change the playback speed or even play the video as picture in picture.
  3. height and width = we give height and width in pixels to adjust the height and width of the video.
  4. autoplay = automatically plays the video as soon as the page loads. we don't use it normally.
  5. loop = it keeps playing the video after it finishes.
  6. poster = we can use posters as thumbnail of our video.
  7. muted = the video will remain muted in start.

The example of video tag.

<video src="source.mp4" controls autoplay loop height="255" width="225" poster="poster.jpg" alt="Poster Image" >

Audio Tag

We use <audio> tag to add the audio in our website.

Attributes of a audio tag.

  1. src = src attribute of the HTML audio tag is primarily designed to point to audio files that are:
    • hosted on the same server as our website.
    • accessible via a direct URL to a audio file.
  2. controls = to add controls on the audio. if controls not added, the audio will not even be displayed let alone be playable.
  3. autoplay = automatically plays the audio as soon as the page loads. we don't use it normally.
  4. loop = it keeps playing the audio after it finishes.
  5. poster = we can use posters as thumbnail of our video.
  6. preload = specifies how a audio file should be loaded when the page loads.('auto', 'metadata', 'none')
  7. The 'preload' attribute can have the following values :
  8. we use preload attribute when we want our browser to fetch data before.
    • none - default value.
    • metadata - only preloads metadata of the audio.
    • auto - automatically gets downloaded.

The example of audio tag.

<audio src="audio.mp3" controls autoplay loop muted >

SVG Files

SVG stands for Scalable Vector Graphics.

SVG files are used to create vector graphics (2D images) that can be resized, scaled, and recolored without losing quality.

SVG files can be created with a text editor or with SVG editors like Adobe Illustrator.

The example of svg file.

<svg height="100" width="100" xmlns="http://www.w3.org/2000/svg"> <circle r="45" cx="50" cy="50" fill="red" stroke="cyan" stroke-width="3"/ > <svg>

to show svg in img attribute we need to add a flag xmlns to the svg file.

IMG TAG

We use image tag to embed images on the website.

we can embed svg images through img tag by providing the img location of the svg file in the src attribute.

Example of img tag.

<img src="image.svg" alt="image-svg" width="45"/>

i-frames in HTML (Embedding other websites on our webpage)

  1. It is a good practice to always include a title attribute for the <iframe>. This is used by screen readers to read out what the content of the iframe is.
  2. Use the height and width attributes to specify the size of the iframe. By default it is in pixels.
  3. <iframe src="demo_iframe.htm" height="200" width="300" title="Iframe Example"></iframe>
  4. <iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Iframe Example"></iframe>
  5. An iframe can be used as the target frame for a link. The target attribute of the link must refer to the name attribute of the iframe
  6. <iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Example"></iframe> <p><a href="https://www.wikipedia.com" target="iframe_a"></a></p>
