ID vs CLASS

ID

ID help us to uniquely identify elements. They are never the same.

IDs are single to a element.

We can also use IDs as links to sections by mentioning the ID in the url we can directly jump to the element containing that ID.

Example of IDs

  1. <section id="section-one" ><section>
  2. ID is an attribute of an element.
  3. In CSS, ID is mentioned as #ID-NAME.

CLASS

Class on the other hand can have multiple elements associated to it.

An element can have multiple classes associated to it. The multiple classes are separated by a space.

Example of Class.

  1. <section class="section-one section-two" id="section-combined" ><section>
  2. CLASS is attribute of an element.
  3. In CSS, class is written as .CLASS-NAME.
