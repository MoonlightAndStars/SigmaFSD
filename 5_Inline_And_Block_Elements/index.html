Inline Elements VS Block Elements

Inline Element

The elements that occupy width that is equal to the content. They do not occupy the whole width and only occupy the width that is required by the content to fill inside.

Examples of Inline Elements

Most commonly used first.

  1. <a> - anchor or hyperlinks
  2. <img> - image.
  3. <span> - generic inline container.
  4. <input> - input field.
  5. <label> - label for a form element.
  6. <strong> - strong emphasis.
  7. <em> - emphasized text.
  8. <br> - line break.
  9. <code> - code snippet.
  10. <b> - bold text.
  11. <i> - italic text.
  12. <u> - underlined text.
  13. <small> - smaller text.
  14. <sub> - subscript.
  15. <sup> - superscript.
  16. <mark> - marked or highlighted text.
  17. <q> - short-inline quotation.
  18. <cite> - citation.
  19. <kbd> - keyboard input.
  20. <samp> - sample output.
  21. <var> - variable in a mathematical expression or programming context.
  22. <time> - time.
  23. <abbr> - abbreviation.
  24. <data> - machine readable translation of context.
  25. <acronym> - acronyms (not supported in HTML 5).

Block Element

The block elements occupy width that is of the viewport.

Note : By changing the display property through CSS we can change the elements to inline from block or to block from inline.

Examples of Block Level Elements

Most commonly used first

  1. <div> : a generic container for flow content.
  2. <p> : paragraph.
  3. <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - headings.
  4. <ul>, <ol>, <li> - unordered list, ordered list and list items.
  5. <form> - a section containing form controls.
  6. <table> - table.
  7. <section> - a standalone section of a document(semantic).
  8. <header> - a container for introductory content or a set of navigational links.
  9. <footer> - footer of a section or a page.
  10. <nav> - a section of a page that contains navigational links.
  11. <article> - a self-contained composition in a document.
  12. <aside> - a section of a page that contains information indirectly related to the main.
  13. <main> - the main content of the document.
  14. <fieldset> - a set of form controls grouped under a common name.
  15. <blockquote> - a block of quote that is a quotation from another source.
  16. <pre> - pre-formatted text.
  17. <canvas> - a container used to draw graphics via javascript.
  18. <dl> - definition list/description list.
  19. <dt> - term in a definition/description list.
  20. <dd> - definition/description in a definition/description list.
  21. <figure> - any content that is referenced from the main content.
  22. <figcaption> - a caption for a <figure> element.
  23. <address> - contact information for the author or owner of the document.
  24. <hr> - a thematic break or a horizontal rule.
  25. <tfoot> - footer of a table.
