CSS can be added in three ways.

  1. Inline
  2. Internal
  3. External

Inline CSS

In inline CSS, we can directly add the style attribute to the tag and give the styling whatever we wish

However, this is not considered a good practice to style elements using inline css.

If we have 1000 lines to style then we will have to give 1000 style values in each line to style them.

They can not be re written by internal or external styles.

Example of Inline Style : <h1 style="color: yellow; background-color : red"> This is a text of yellow color with red background styled using inline css.<h1>

Internal CSS

We use internal css when our project is small.

In internal css, a style tag is created in the head section of the html and all the styling is written there.

Internal CSS is better than inline CSS and we can bulk edit our style.

The drawback of internal css is that once we have a huge number of code and we have to style the webpage then because of styling our webpage our file will keep on growing and this will create problem.

In the head section we create <style> </style> and write out styling between this style tag.

External CSS

A separate CSS file is created by the name of style.css and in the head section we link the css by link tag using href to the stylesheet.
