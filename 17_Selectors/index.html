CSS Selectors

We use selectors to target specific HTML elements using CSS.

Types of CSS Selectors

  1. Universal Selector:

    It selects all elements using *. Example:

    * { margin: 0; padding: 0; box-sizing: border-box; }
  2. Element Selector:

    Targets an HTML element directly (e.g., p {}, h1 {}).

  3. Class Selector:

    Targets elements with a specific class. Example:

    .btn { background-color: blue; }
  4. ID Selector:

    Targets a specific element using #id. Example:

    #header { font-size: 24px; }
  5. Group Selector:

    Applies same styles to multiple elements. Example:

    h1, h2, p { color: black; }
  6. Child Selector (>):

    Targets direct children only. Example:

    div > p { color: red; }
  7. Descendant Selector (space):

    Targets all nested elements. Example:

    div p { font-size: 14px; }
  8. Adjacent Sibling Selector (+):

    Selects next immediate sibling. Example:

    h1 + p { color: gray; }
  9. General Sibling Selector (~):

    Selects all siblings. Example:

    h1 ~ p { color: blue; }
  10. Attribute Selector:

    Targets elements with a specific attribute. Example:

    input[type="text"] { border: 1px solid black; }
  11. Pseudo-class Selector:

    Applies styles on a specific state. Example:

    a:hover { color: red; }
  12. Pseudo-element Selector:

    Targets parts of an element. Example:

    p::first-letter { font-size: 20px; }
